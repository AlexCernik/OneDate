What is this?

The One Piece API is a RESTfull API based on the Anime show One Piece. You will have access to about hundreds of character's images,reward and rank's.

Check out documentation to get started

Who are you?

We are Dominyel.YoY, a guy who likes to develop things and AlexCernik, the "One Piece" and hardcode fan.

Why did you build this?

Because we were reallu interesred in the idea of writing an open source project and also because One Piece is our favorite show at the moment, so why not?

Technical stuff?

We use Django and to serve the API. All the data is format json and the entire project is hosted on Heroku and Alwaysdata.

How can I contribute to the project

You can help us to keep the project alive and you can also contribute on Github.
